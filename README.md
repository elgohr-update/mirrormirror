# [Mirror Mirror](https://mirrormirror.florianporada.com/)

A VR experience to get more in touch with your digital avatar.

## Usage

### Installation

Install the dependencies

```sh
yarn install
```

### Serve

To serve in the browser - Runs webpack-dev-server

```sh
yarn start
```

### Build

Compile and build

```sh
yarn run build
```
